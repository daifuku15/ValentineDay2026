# 💝 Valentine's Day Heart App

バレンタインデー用のインタラクティブなWebアプリです。
クリックやタップをするとハートマーク（❤️）が飛び出します！

## 🌟 機能

- クリック/タップでハートが飛び出す
- 8種類のハート絵文字がランダムに表示
- 美しいグラデーション背景（5秒ごとに変化）
- スマホ・タブレット対応
- ページ読み込み時の自動アニメーション

## 🚀 GitHub Pagesでの公開方法

### 1. リポジトリの作成
```bash
# 新しいリポジトリを作成（GitHubで）
# 例: valentine-heart-app
```

### 2. ファイルのアップロード
```bash
# ローカルでの作業
git init
git add index.html README.md
git commit -m "Initial commit: Valentine's Heart App"
git branch -M main
git remote add origin https://github.com/あなたのユーザー名/valentine-heart-app.git
git push -u origin main
```

### 3. GitHub Pagesの有効化
1. GitHubのリポジトリページを開く
2. 「Settings」タブをクリック
3. 左サイドバーの「Pages」をクリック
4. 「Source」で「main」ブランチを選択
5. 「Save」をクリック

### 4. 公開URLの確認
数分後、以下のようなURLでアクセス可能になります：
```
https://あなたのユーザー名.github.io/valentine-heart-app/
```

## 📱 使い方

1. 上記のURLにアクセス
2. 画面をクリック/タップ
3. ハートが飛び出す！❤️
4. リンクをシェアして、大切な人に送ろう

## ✨ カスタマイズ

`index.html`を編集することで、以下をカスタマイズできます：

- メッセージの変更（`<h1>`と`<p class="subtitle">`）
- 色の変更（`gradients`配列）
- ハートの種類（`hearts`配列）
- アニメーション速度（CSSの`animation`）
